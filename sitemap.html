🤖 구글 검색엔진을 위한 XML 사이트맵

위 HTML 사이트맵은 사람이 보기 위한 것이에요. 구글이 사이트를 빠르게 색인하려면 XML 형식의 사이트맵도 필요해요. GitHub Pages에 배포한다면 루트 디렉터리에 아래 파일을 만들어 주세요.

<?xml version="1.0" encoding="UTF-8"?> <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"> <url> <loc>https://aun-jy.github.io/vibe-toolkit/</loc> <changefreq>weekly</changefreq> <priority>1.0</priority> </url> <url> <loc>https://aun-jy.github.io/vibe-toolkit/favicon-sample.html</loc> <changefreq>monthly</changefreq> <priority>0.9</priority> </url> <url> <loc>https://aun-jy.github.io/vibe-toolkit/google-analytics.html</loc> <priority>0.9</priority> </url> <!-- ... (총 21개 URL 등록됨) --> </urlset>
💡 파일명을 sitemap.xml로 저장하고, Google Search Console에서 제출하면 더 빠르게 색인돼요. 자세한 방법은 사이트맵 만들기 가이드에서 단계별로 안내해 드려요.
