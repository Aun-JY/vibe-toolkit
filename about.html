왜 바이브툴킷을 만들었나요?

ChatGPT나 Claude로 코드를 받아도 "이걸 어디에 어떻게 쓰지?" 하는 순간, 누구나 한 번쯤 막막했을 거예요.

😰

AI가 코드를 줬는데…

HTML, CSS, JS를 처음 배우는 분들이 AI에게 코드를 받고 나서 "이 코드를 어디다 붙여 넣어야 하지?", "왜 안 되지?" 하며 막히는 경우가 너무 많아요.

🔍

구글 검색은 너무 어렵고

검색을 해봐도 Stack Overflow의 영어 답변, 3년 전 블로그 글… 초보자 눈높이에 맞는 한국어 실전 자료가 너무 부족했어요.

💡

직접 만들어 써봐야 안다

코드는 읽는 것만으론 절대 익혀지지 않아요. 직접 수정하고, 눈으로 확인하고, 내 사이트에 적용해봐야 진짜 배우게 돼요.

🎯

그래서 만들었어요

복붙만 해도 동작하는 코드, 실시간으로 수정해볼 수 있는 에디터, 한국어로 친절하게 설명하는 가이드. 이게 바이브툴킷이에요.

바이브 코딩이 뭔가요?

2025년 가장 핫한 코딩 방식. AI와 대화하면서 실시간으로 코드를 만들어가는 방법이에요.

🤖

AI + 나 = 바이브 코딩

"바이브 코딩(Vibe Coding)"은 2025년 Andrej Karpathy가 처음 명명한 개념으로, 개발자가 AI(ChatGPT, Claude 등)와 자연어로 대화하면서 코드를 작성하는 방식이에요. 완벽한 코드 문법을 몰라도, 원하는 기능을 말로 설명하면 AI가 코드를 만들어줘요.

1

💬 AI에게 요청하기

ChatGPT나 Claude에게 "브라우저 탭에 보이는 파비콘 SVG 코드 만들어줘, 보라색 배경에 V 글자"처럼 원하는 것을 말로 설명해요.

2

📋 코드 받아서 이해하기

AI가 만들어준 코드를 받아요. 바이브툴킷의 해설을 참고해서 각 속성이 무슨 의미인지 파악해요.

3

🔧 에디터에서 수정해보기

바이브툴킷 에디터에 코드를 붙여넣고 색상, 크기, 모양을 직접 바꿔보며 결과를 실시간으로 확인해요.

4

🚀 내 사이트에 적용!

완성된 코드를 내 HTML 파일에 붙여넣으면 끝! 가이드대로만 따라 하면 누구든 적용할 수 있어요.

바이브툴킷이 지키는 원칙

도구를 만들면서 항상 이 세 가지를 최우선으로 생각해요.

🆓

완전 무료

모든 도구와 코드는 영원히 무료예요. 회원가입 없이 바로 사용 가능해요.

🇰🇷

한국어 우선

영어 자료가 넘치는 세상. 한국어로 쉽고 친절하게 설명하는 곳이 더 필요해요.

✂️

복붙하면 바로 동작

예제 코드를 복사해서 붙여넣기만 해도 실제로 동작해야 한다고 믿어요.

👨‍💻
